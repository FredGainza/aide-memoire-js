WHILE (TANT QUE)
L'instruction while permet de créer une boucle qui s'exécute tant qu'une condition de test est vérifiée.

	if(i <= 1){
		console.log(i + " tour");
	}else{
		console.log(i + " tours");
	}
	i++;
}

La condition est évaluée après avoir exécuté une itération de boucle,nce qui fait que cette boucle sera exécutée au moins une fois.

    • scroll : défilement de la fenêtre
    • resize : redimensionnement

  • 4. Les évènements : FORM (formulaire)
    • change : pour les éléments <input>, <select>, <textarea>
    • submit : l'envoi / soumission du formulaire
    • input : pour capter la saisie d'un utilisateur sur un champ <input>

$(selecteur).action()

$(selecteur).methode1().methode2().methode3();

Méthodes pour affiner la sélection : has(), filter(), not(), first(), last() et eq()

Méthode has()
Ne sélectionne que les éléments d’une sélection initiale qui possèdent des éléments enfants correspondant au sélecteur passé en argument de cette fonction
* exemple : sélectionner uniquement les éléments p qui contiennent un span 
 $("p").has("span");

Méthode filter()
Ne sélectionne que les éléments qui satisfont au sélecteur passé en argument de cette fonction
* exemple : ne sélectionner que les éléments span d’une page qui possèdent un certain attribut class 
$("span").filter(".souligne")

Méthode not()
Soustraire d’une sélection de départ des éléments qui répondent à un certain critère
* exemple : supprimer les éléments span possédant un certain attribut class d’une sélection 
$("span").not(".souligne")

Méthodes first(), last() et eq()
Ne sélectionne que le premier, le dernier, ou le ième élément d'une sélection de départ
exemple : 
$("span").first()   // sélectionne le 1er span
	$("span").last()    // sélectionne le dernier span
	$("span").eq(1)     // sélectionne le 2éme span

Accéder à l'élément parent ou aux ancetres d'un élément en jQuery

Méthode parent()
Permet d’accéder à l’élément parent (c’est-à-dire à l’ancêtre direct) de l'élément sélectionné;
- On peut lui passer un sélecteur en argument, ce qui va filtrer la sélection :
- On ne va accéder au parent que s'il correspond au sélecteur 
$("li").parent()            // cible le parent direct de chaque li 
$("li").parent("#parents")  // cible le parent direct de chaque li ssi il possède l'id="parents"

Méthode parents()
Permet d’accéder aux éléments enfants (descendants directs) d’un élément html;
- Fonctionne comme parent() : filtre la sélection par le sélecteur passé en argument 
$("ul").parents()       // cible les ancetres des ul
$("ul").parents("div")  // cible les ancetres des ul ssi ils sont dans une div

Accéder aux enfants ou descendants d'un élément en jQuery

Méthode children()
Permet d’accéder à l’élément parent (c’est-à-dire à l’ancêtre direct) de l'élément sélectionné
$(".conteneur").children()      // cible les enfants des éléments contenant class="conteneur"
$(".conteneur").children("h1")  // cible que les h1 enfants des éléments contenant class="conteneur"

Méthode find()
Permet d’accéder aux descendants d’un élément html
Si sélecteur passé en argument, il filtre la sélection 
$(".souligne").find()      // cible tous les descendants des éléments contenant class="souligne"
$(".souligne").find("h1")  // cible que les h1 descendants des éléments contenant class="souligne"
rq: pour avoir tous les descendants d'un élément, on passe * en argument 
$("#titre").find("*")       // cible tous les descendants de l'élément id="titre"

Accéder aux éléments frères d'un élément en jQuery

Remarque : pour tous, possible de passer un sélecteur en argument pour filtrer la sélection

Méthode siblings()
Permet d'accéder à l'ensemble des éléments frères d’un élément sélectionné
$(selector).siblings(filter)

Méthode next()
Permet d'accéder à l'élément frère suivant immédiatement l'élément sélectionné
$(selector).next(filter)

Méthode prev()
Permet d'accéder à l'élément frère précédent immédiatement l'élément sélectionné
$(selector).prev(filter)

FONCTIONNEMENT DES METHODES GETTERS ET SETTERS jQuery

Contrairement à JS qui a des getters et des setters (ex getAttribute & setAttribute), les méthodes en jQuery vont occuper les 2 fonctions :
  • - fonction de setter si la méthode n'a pas d'argument
  • - fonction de getter si la méthode possède un argument
Remarque : si on sélectionne plusieurs éléments :
  • - si fonction de setter : tous les éléments sont pris en compte
  • - si fonction de getter : seul l'info du 1er élément sera renvoyé
  • exception pour text() qui renverra les valeurs de tous les éléments

Les differentes méthodes usuelles

Méthode html()
Permet d’obtenir le contenu HTML du premier élément d’une sélection ou de définir le contenu HTML de tous les éléments de la sélection;
$("h1").html()  // html() comme getter
$("h2").html(Nouveau titre)   // html() comme setter

Méthode text()
Permet d’obtenir ou de définir les contenus textuels de tous les éléments d’une sélection;
Remarque : A la différence de la méthode html(), seul le contenu textuel va pouvoir être récupéré et on ne va pouvoir également définir que du contenu textuel dans nos éléments.

Méthode attr()
Permet d’obtenir la valeur d’un attribut du premier élément d’une sélection ou de définir un ou plusieurs attributs pour tous les éléments de la sélection;

Méthode prop()
Permet d’obtenir la valeur d’une propriété du premier élément d’une sélection ou de définir une ou plusieurs propriétés pour tous les éléments de la sélection;

Méthode val()
Permet d’obtenir la valeur (le contenu de l’attribut value) du premier élément d’une sélection ou de définir une valeur pour tous les éléments de la sélection;

Méthode css()
Permet d’obtenir la valeur d’une propriété CSS pour le premier élément d’une sélection ou de définir les valeurs d’une ou de plusieurs propriétés CSS pour tous les éléments de la sélection;

Insérer du contenu en tant que premier (prepend()) et dernier enfant (append()) d’un élément cible.

Elle ne va pas renvoyer la liste des éléments supprimés
$(content).replaceAll(selector)

$(selector).replaceWith(content,function(index))

class Feu{
	// Permet de créer les paramètres par défaut
	constructor(nom)
	{
		this.nom        = nom;
		this.puissance  = Math.floor(Math.random() * Math.floor(60));
		this.vie        = 200;
	}

	// Methode pour attaquer le pompier
	bruler(pompier)
	{
		if(this.vie > 0){
			// Je stocke dans une variable la vie du pompier
			// Je soustrais la puissance du feu et la brulure() à la vie du pompier
			let resultatCombat = pompier.vie - this.puissance - this.brulure();
			pompier.vie = resultatCombat <= 0 ? 0 : resultatCombat; 
			// ecriture ternaire
			// si resultatCombat est inferieur ou égal à 0, ALORS (?) je mets 0 à la vie du pompier 
			// SINON (:) on met resultatCombat à la vie
			
			return resultatCombat;
		}
		return "";
	}

	{

		// que je reprends dans l'attaque bruler()

			return 0;
		}
	}

	// que l'on souhaite

	{
		return Math.floor(Math.random() * Math.floor(max));
	}
